# 疯狂园丁 V2 手感与表现升级规格

## 目标

V2 的目标是在不重做整体架构的前提下，让第一版原型更像一个能顺畅试玩的横版动作游戏。重点是中文化、镜头跟随、关卡节奏、枪托挥杆表现、尸体击飞、数值调校和 Boss 信息展示。

## 范围

本次升级覆盖以下内容：

- 游戏界面中文化。
- 捡起强化时，在人物旁边弹出中文强化提示。
- 镜头根据人物移动跟随，而不是让画面感觉固定在地面上。
- 通关模式清关后弹出“下一关”按钮，点击后回到出生点进入下一关。
- 僵尸移动速度降低，血量略微提高。
- 右键枪托表现为明显的高尔夫挥杆动作。
- 普通僵尸被枪托击杀后留下尸体，尸体被打飞，5 秒后消失。
- 降低空中向下开枪的后坐力高度。
- 减少散弹子弹数量。
- 通关模式显示剩余怪物数量；无尽模式不显示剩余怪物。
- 任意模式出现 Boss 时，屏幕顶部显示 Boss 大血条。
- Boss 关中，先打完 Boss 前的小怪，再在离玩家中等距离的位置生成 Boss。

不包含：

- 新 Boss 类型。
- 新敌人类型。
- 音效系统。
- 存档或持久升级。
- 精细像素/贴图美术。

## 中文化

所有玩家可见文本改为中文，包括：

- 主菜单按钮。
- HUD 标签。
- 控制提示。
- 强化名称。
- 换弹状态。
- 通关提示。
- 下一关按钮。
- 模式名称。
- Boss 血条名称。

推荐文本：

- “关卡模式”
- “无尽模式”
- “生命”
- “弹药”
- “换弹中”
- “准备就绪”
- “第 X 关”
- “第 X 波”
- “剩余怪物”
- “击杀”
- “分数”
- “强化”
- “下一关”
- “获得：穿透弹”

## 强化拾取提示

玩家拾取强化时，在角色附近生成一条短暂浮动文字。

行为要求：

- 文字显示在玩家头顶或身体旁边。
- 内容格式为“获得：强化名”。
- 持续约 1.2 秒。
- 文字向上漂浮并逐渐淡出。
- 不遮挡 HUD。

强化中文名：

- 穿透弹
- 大扩散
- 快速换弹
- 强后坐力
- 长枪托

## 镜头跟随

镜头应跟随玩家横向移动，让玩家保持在屏幕中偏左的位置。地面、敌人、掉落、撤离点和尸体都要按镜头偏移绘制。

要求：

- 玩家向右推进时，背景和地面元素随镜头平滑移动。
- 玩家靠近地图左端或右端时，镜头不会超过世界边界。
- 镜头跟随应比瞬间锁死更平滑，可使用插值。
- 鼠标瞄准的世界坐标必须继续正确。

## 通关流程

通关模式不再让玩家直接碰撤离点进入下一关。清关后应弹出“下一关”按钮或居中面板。

普通关：

1. 生成本关小怪。
2. 显示剩余怪物数量。
3. 小怪全部被击杀后，弹出“下一关”按钮。
4. 点击按钮后进入下一关。
5. 玩家回到出生点，弹药和换弹状态重置。

Boss 关：

1. 先生成 Boss 前的小怪。
2. 小怪全部被击杀后，在离玩家中等距离的位置生成 Boss。
3. Boss 出现后显示顶部 Boss 大血条。
4. Boss 被击杀后弹出“下一关”按钮。
5. 点击按钮后进入下一关。

“中等距离”定义：

- Boss 生成在玩家面朝方向约 520 到 680 像素处。
- 如果玩家靠近地图边缘，生成点应被限制在世界边界内。
- Boss 不应直接贴脸生成。

## 无尽模式流程

无尽模式保持自动刷怪，不显示“剩余怪物”。HUD 仍显示生命、弹药、换弹、波次、击杀、分数和强化。

Boss 在无尽模式出现时也显示顶部大血条。无尽模式不出现“下一关”按钮。

## 敌人数值调校

僵尸整体要跑慢一点，血量稍微增加。

建议调校：

- 普通僵尸速度降低约 25%，血量提高约 25%。
- 快跑僵尸仍然明显快于普通僵尸，但速度降低约 20%，血量略微提高。
- 胖僵尸速度略降，血量提高约 15%。
- Boss 速度可略降，血量保持或略增。

目标感觉：

- 玩家有时间看见僵尸接近。
- 散弹枪不再过于扫屏。
- 枪托仍然是处理普通僵尸的高风险秒杀技。

## 武器数值调校

散弹子弹数量减少，后坐力降低。

建议调校：

- 每次开枪弹丸数从 8 减到 6。
- 空中向下开枪的额外后坐力降低约 35%。
- 强后坐力强化仍然能明显飞得更高，但不应高到脱离战斗节奏。
- 散弹总伤害可通过僵尸血量和弹丸伤害一起微调，确保普通僵尸不会被远距离随便秒杀。

## 枪托挥杆与尸体

右键攻击要表现为高尔夫挥杆，而不只是一个静态弧线。

挥杆要求：

- 挥杆有短暂动画进度。
- 枪从身后/侧后方向前扫出。
- 挥杆弧线随动画进度展开。
- 命中普通僵尸时有明显击飞反馈。

尸体要求：

- 被枪托击杀的普通僵尸不立即消失。
- 生成一个尸体对象，沿挥杆方向飞出，并带向上的抛物线速度。
- 尸体受重力影响，落地后可滑行/停下。
- 尸体保留 5 秒后消失。
- 尸体不再造成伤害，也不阻挡玩家或子弹。

普通散弹击杀的僵尸可以继续直接消失或使用较小的消失特效；本次尸体飞出重点服务枪托击杀。

## Boss 大血条

任意模式只要存在 Boss，就在屏幕顶部显示 Boss 大血条。

要求：

- 血条居中显示，宽度明显大于普通敌人血条。
- 显示 Boss 名称。
- 血量按当前 Boss health / maxHealth 显示。
- Boss 死亡后血条消失。
- 如果未来同屏多个 Boss，V2 只显示第一个 Boss 即可。

## HUD

通关模式显示：

- 生命
- 弹药
- 换弹状态
- 当前关卡
- 剩余怪物
- 击杀
- 分数
- 当前强化

无尽模式显示：

- 生命
- 弹药
- 换弹状态
- 当前波次
- 击杀
- 分数
- 当前强化

无尽模式不显示剩余怪物。

## 架构影响

建议沿用现有模块：

- `src/core/constants.js`：调整中文名称和数值。
- `src/core/state.js`：如需要，增加等待下一关、尸体、浮动文字等初始字段。
- `src/core/systems.js`：实现尸体、浮动文字、Boss 延迟生成、下一关状态、镜头平滑、数值效果。
- `src/render.js`：中文 HUD、Boss 大血条、下一关按钮提示、挥杆动画、尸体绘制、浮动文字。
- `src/main.js`：处理“下一关”按钮点击。
- `src/styles.css` / `index.html`：中文菜单、下一关按钮、必要样式。
- `tests/state.test.mjs`：覆盖关键行为。

## 测试计划

自动测试至少覆盖：

- 中文强化名称可用于拾取提示。
- 拾取强化会生成浮动提示。
- 枪托击杀普通僵尸会生成 5 秒尸体。
- 枪托不会秒杀 Boss。
- Boss 关先清小怪，随后生成 Boss。
- Boss 生成位置离玩家为中等距离并受世界边界限制。
- 清关后进入等待下一关状态。
- 点击/调用下一关推进后玩家回出生点。
- 无尽模式不显示剩余怪物所依赖的状态。
- 弹丸数减少到 6。
- 后坐力降低后仍可向上弹起。

人工验证：

- 菜单和 HUD 全中文。
- 捡强化时人物旁边出现“获得：...”。
- 镜头随人物横向移动。
- 通关后出现“下一关”按钮。
- Boss 前小怪清完后 Boss 在中等距离生成。
- Boss 顶部大血条出现并随伤害减少。
- 右键挥杆像高尔夫挥杆，尸体飞出并 5 秒后消失。
- 僵尸速度更慢，血量略高。
- 后坐力不再飞得过高。

## 成功标准

V2 成功时，玩家能明显感觉到：

- 游戏界面已经中文化。
- 关卡模式更像正式流程：清怪、Boss、下一关。
- 镜头跟着角色推进。
- 枪托像高尔夫挥杆，击杀后尸体飞出去。
- 强化拾取反馈清楚。
- Boss 信息清楚。
- 数值更稳，不再子弹过多、飞得过高、僵尸冲得太快。
